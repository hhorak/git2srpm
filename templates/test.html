git2srpm

Simple tool to generate SRPM from git repository and using lookaside cache from Fedora.

Generate

Enter valid git repository url

1. Submit git repo

Enter any public git repository that includes one SPEC file.

2. See link

From the git repository a SRPM is generated and link to this file is provided.

3. Use in copr

The provided link can then be used in copr
