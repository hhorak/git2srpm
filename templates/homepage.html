git2srpm

Simple tool to generate SRPM from git repository and using lookaside cache from Fedora.

  • Generate
  • Search

Generate

Enter a valid git repository url that contains SRPM sources

1. Submit git repo

Enter link to any public git repository that includes one SPEC file and other SRPM sources.

2. Get SRPM link

This application generates a SRPM using lookaside cache from Fedora and gives you a link to SRPM.

3. Use link in copr

The provided link can then be used in copr project to build your ad-hoc package.
